# Foreman ASCII Mascots

Üç farklı boyutta, terminal kullanımı için hazır. Boyutları seçerken:

- **BIG**: boot banner için (program ilk açıldığında), README.md başında
- **MEDIUM**: yardım ekranı, about sayfası, lansman banner'ı
- **MINI**: terminal prompt'larında, log satırlarında, küçük her yerde

Tümünde turuncu (ANSI 208 / true color #FF8C42) renkli kullanmalısın.
chalk veya ink ile renklendirilir.

═══════════════════════════════════════════════════════════════════════
  RENK KULLANIMI (Ink / chalk)
═══════════════════════════════════════════════════════════════════════

Karakter parçaları farklı renklerde olabilir:

- Baret ([F] ve dome çizgileri): #FF8C42 (orange)
- Vücut/yüz çizgileri: #8B5A2B (brown) veya default fg
- Yelek (VEST yazısı ve ===): #FF8C42 (orange)
- Gözler / dişler: default fg
- "F" rozeti içi: bold + #FFB52E (yellow)

Ink JSX örneği:

  <Box flexDirection="column">
    <Text color="#FF8C42">{`       ___[F]___`}</Text>
    <Text>{`      /  _____  \\`}</Text>
    <Text>{`     |__/ o   o \\__|`}</Text>
    <Text>{`        |  \\_/  |`}</Text>
    <Text>{`       /|_______|\\`}</Text>
    <Text color="#FF8C42">{`      / |==VEST=| \\`}</Text>
    <Text color="#FF8C42">{`     /__|=======|__\\`}</Text>
    <Text>{`        |_______|`}</Text>
  </Box>


═══════════════════════════════════════════════════════════════════════
  NOTLAR
═══════════════════════════════════════════════════════════════════════

1. Karakterler **monospace font'ta** doğru hizalanacak şekilde tasarlandı.
   Terminal'lerin hepsi monospace olduğu için sorun yok.

2. Backslash (\) karakterleri JavaScript/TypeScript string'lerinde
   escape edilmeli: \\.

3. Eğer ASCII karakter desteklemeyen bir terminal varsa (çok nadir),
   fallback olarak sadece "🦫 Foreman" yazılabilir.

4. Daha şirin/profesyonel bir versiyon istersen, kunduzu yandan göster,
   kuyruğunu göster (kunduzun signature feature'u). Aşağıda denemesi:


       ___[F]___
      /  _____  \
     |__/ o   o \__|        _
        |  \_/  |          | |
       /|_______|\         | |____,
      / |==VEST=| \   ____/      /   <- flat tail
     /__|=======|__\ /_________/
        |_______|

5. README için emoji + ASCII karması da güzel olur:

   🦫 **Foreman** — your agent guardian
   `[F]` your AI crew's foreman
