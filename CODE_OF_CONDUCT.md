# Code of Conduct

Foreman follows the [Contributor Covenant, version 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/) as its code of conduct. The canonical text and its translations live at that URL — treat it as the authoritative version for this project.

In short: contributors, maintainers, and community members are expected to make participation in Foreman a respectful and welcoming experience for everyone, regardless of background or identity.

## Scope

This applies within all project spaces — the GitHub repository, issues, pull requests, discussions, and any other channel that represents the project — and when an individual is officially representing the project in public.

## Reporting

If you experience or witness behavior that breaks the Contributor Covenant, please report it privately to **fatihtuzlu07@gmail.com**. Reports are handled confidentially. The maintainer will review, follow up, and take whatever action is needed — from a private warning to a permanent ban — based on the [Enforcement Guidelines](https://www.contributor-covenant.org/version/2/1/code_of_conduct/#enforcement-guidelines) section of the Contributor Covenant.
